Here is a basic example using the Arduino_GFX_Library and CST816S Touch library to get the Waveshare ESP32-S3 2" LCD display running. Please copy files into their respective places in VSCode/PlatformIO and hopefully this will get you started.

The board and wiki are found here:

https://www.waveshare.com/esp32-s3-touch-lcd-2.htm?sku=29667

https://www.waveshare.com/wiki/ESP32-S3-Touch-LCD-2

Link to Arduino GFX library https://github.com/moononournation/Arduino_GFX

Link to CST816S Library https://github.com/fbiego/CST816S.git
